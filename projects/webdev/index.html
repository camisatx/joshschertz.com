Here are some of the websites that I have designed and developed. Each of them is unique, with differing levels of work required for them. Some I built from scratch, others used a web framework (ReactJS, Flask, Jekyll), and others were designed in SquareSpace.

Pickle Rick: A silly website based on the Rick and Morty clip of Rick turning himself into a pickle. The amusement comes from Rick revealing he is the pickle, so I turned that into a command line surprise. When you type "flip-pickle" or "flip" into the mock-terminal, an asciiart image of Rick the pickle is shown. This was built in ReactJS using the terminal-in-react library. It made it as high as 16 on the home page of Hacker News, receiving over 17,000 visitors during the first 24 hour period. Link

Consens: Your personal fashionista. A custom ReactJS app using Flask for backend API, Postgres for the database, Docker to manage deployments, Backblaze for image storage, Cloudflare for CDN and image processing, and Amazon Mechanical Turk for worker tasks. Link
